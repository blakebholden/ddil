## Tools Required

Gather these before starting assembly:

- [ ] Phillips head screwdriver (#1 and #2)
- [ ] Anti-static wrist strap
- [ ] Cable ties (assorted sizes)
- [ ] Label maker or masking tape + marker
- [ ] Small flashlight
- [ ] Multimeter (optional, for power verification)
- [ ] Isopropyl alcohol + microfiber cloth (for thermal paste if needed)

---

## Workspace Preparation

- [ ] Clear workspace (minimum 4ft x 3ft)
- [ ] ESD-safe surface or mat
- [ ] Good lighting
- [ ] Power strip with surge protection for testing
- [ ] Network connection for initial software downloads
- [ ] Laptop/workstation for configuration

After unboxing, organize components into groups:

### Group C: Network

### Group D: Accessories

- Power cables
- Pelican case + foam
